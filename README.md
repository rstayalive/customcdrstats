# Freepbx Custom CDR stats

Requirements:
FreePBX 16/17 with PJSIP

Simple setup: Go to Admin > Module Admin > Upload Modules to install the module.
get latest version url from https://github.com/rstayalive/customcdrstats/releases/latest
